# 麻将馆记账本

面向麻将馆的收入与支出记账工具，单页静态网页应用，数据保存在浏览器本地。

## 功能

- **仪表盘**：今日与本月的收入、支出、利润概览
- **流水**：按日分组的月度记账明细
- **月报**：按类别统计的收入与支出图表
- **数据管理**：JSON 导出/导入、每日自动备份

## 快速开始

无需安装依赖。在仓库根目录启动静态服务：

```bash
python3 -m http.server 8080
```

浏览器访问：http://localhost:8080/index.html

也可直接双击或用浏览器打开 `index.html`。

## 技术说明

- 单文件：`index.html`（内联 CSS 与 JavaScript）
- 存储：浏览器 `localStorage`
- 无后端、无构建、无网络依赖（加载后可离线使用）
