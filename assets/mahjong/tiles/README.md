# Mahjong tile images

This directory stores the project's canonical 42 Mahjong tile face images.

- Source site: `https://majiang.world`
- Remote JPG template: `https://el.eeurl.com/jpg/mj/s2/{code}.jpg`
- Local JPG template: `assets/mahjong/tiles/{code}.jpg`
- Mapping file: `mahjong_tiles.json`

Codes follow the `majiang.world` standard:

- `a1`-`a9`: characters / 万子
- `b1`-`b9`: bamboo / 索子
- `c1`-`c9`: dots / 筒子
- `d1`-`d4`: winds / 东南西北
- `e1`-`e3`: dragons / 中发白
- `f1`-`f8`: flowers / 春夏秋冬梅兰竹菊
